# DinoRunWithArduino
Using an Arduino and 360 servo motor to automate playing the dino game on google.

link up an LDR to a 10K ohm resistor and insert it into the A0 pin
link up a servo motor brown - GND, red - 5V, yellow - 8pin

tape the servo motor to the space bar and stick the ldr to the screen with around 1-3 cm away from the dinosaur
